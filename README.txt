=============Usage=================
"node appPhaser.js" to start the server locally.
"localhost:3000" in browser to visit the site locally.

===========Requirements============
- Easystar v0.4.3(https://easystarjs.com/)
- Phaser 3 v3.22.0 (https://github.com/photonstorm/phaser/releases/tag/v3.22.0)
- Bootstrap v4.4.1 CSS (https://stackpath.bootstrapcdn.com/bootstrap/4.4.1/css/bootstrap.min.css)
- Bootstrap v4.4.1 JS (https://stackpath.bootstrapcdn.com/bootstrap/4.4.1/js/bootstrap.min.js)
- Jquery v3.4.1 (https://code.jquery.com/jquery-3.4.1.slim.min.js)
- Popper v1.16.0 (https://cdn.jsdelivr.net/npm/popper.js@1.16.0/dist/umd/popper.min.js)
